Tron Script

TronScript es una herramienta de código abierto para limpiar y optimizar sistemas Windows. Realiza tareas como eliminar malware, actualizar software, limpiar archivos temporales y optimizar el registro. Debe usarse con precaución y respaldar datos antes de usarlo, ya que algunas acciones pueden ser irreversibles.
